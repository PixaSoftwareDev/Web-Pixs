# Pixs — Landing

Landing page 3D para Pixs, estudio de desarrollo de software.
Stack: **Next.js 14 + TypeScript + Tailwind + Three.js + GSAP**.

## Levantar el proyecto

```bash
npm install
npm run dev
```

Abrí http://localhost:3000.

## Estructura

Ver [`CLAUDE.md`](./CLAUDE.md) — ahí está todo el contexto del proyecto: stack, paleta, secciones, dónde cargar el logo y las fotos, comandos y reglas para trabajar con Claude.

## Cargar tus assets

- Logo → `public/assets/logo/`
- Fotos de proyectos → `public/assets/portfolio/`
- (Opcional) modelo 3D → `public/assets/models/`

## Build

```bash
npm run build
npm start
```
