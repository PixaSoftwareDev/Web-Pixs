# Pixs — Landing

Landing page para Pixs, estudio de desarrollo de software.
Stack: **Next.js 14 + TypeScript + Tailwind + GSAP**.

## Levantar el proyecto

```bash
npm install
npm run dev
```

Abrí http://localhost:3000.

## Estructura

Ver [`CLAUDE.md`](./CLAUDE.md) — ahí está todo el contexto del proyecto: stack, paleta, secciones, comandos y reglas para trabajar con Claude.

## Build

```bash
npm run build
npm start
```
