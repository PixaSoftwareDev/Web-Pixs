# Portfolio

Cargá acá las imágenes de los proyectos para la sección Portfolio.

Convención de nombres:

- `proyecto-1.webp`
- `proyecto-2.webp`
- `proyecto-3.webp`
- `proyecto-4.webp`

Recomendaciones:

- **Formato**: WebP (o PNG/JPG si no podés convertir).
- **Tamaño**: 1600 x 1000 px aprox. (ratio 16:10).
- **Peso**: bajo 300 KB por imagen idealmente.

Una vez cargadas, editá los títulos/descripciones en `src/lib/content.ts` y descomentá/agregá el `<Image>` en `src/components/sections/Portfolio.tsx` reemplazando el placeholder visual.
