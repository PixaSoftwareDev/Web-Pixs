# Logo

Cargá acá los archivos del logo de Pixs:

- `logo.svg` — versión completa (isotipo + texto). **SVG ideal.**
- `logo-mark.svg` — solo isotipo (para favicon, navbar compacta).
- `logo-wordmark.svg` — solo el texto.
- `logo-light.svg` *(opcional)* — versión para fondo claro.

Si solo tenés PNG, también sirve: `logo.png` (mínimo 1024px de ancho, fondo transparente).

Después decile a Claude: *"el logo ya está en `public/assets/logo/`, usalo en el Navbar y el Footer"*.
